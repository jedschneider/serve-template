h1. My Serve Templating Project

just a skelton project directory to use and quickly create projects to used with the serve web server.

to use:

@gem install serve@

I had to install mongrel

@gem install mongrel@

and then serve it

@cd myproject@

@serve@

and navigate to localhost:4000

to find out more about serve visit the 'project page':http://github.com/jlong/serve
